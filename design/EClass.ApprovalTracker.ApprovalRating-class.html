Class ApprovalRating

source code

ApprovalRating is a value which represents the approval of the presentation at this very instant.

If an ApprovalRating is owned by a Student, it represents that student's rating.

If an ApprovalRating is owned by a Presenter, it represents the class average.

Author: Josh Terrell jmterrel@calpoly.edu

SetValue(self, percent)

source code 

Set the the value of this approval rating to a percentage between [0, 1].

A side effect of calling this method is that all listeners added via AddListener are notified that the value has changed.

Parameters:
  • percent - the percentage to set this rating to

Postcondition: self.__rating == percent
