Class ApprovalRating

source code

ApprovalRating is a value which represents the approval of the presentation at this very instant.

If an ApprovalRating is owned by a Student, it represents that student's rating.

If an ApprovalRating is owned by a Presenter, it represents the class average.

Author: Josh Terrell
