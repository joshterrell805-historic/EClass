Class EClass

source code

EClass is a colleciton of all the highest level models in the EClass program including (but not limited to) the presentation, layer manager, and logged in user.

Login(self, username, password)

source code 

Attempt to login the user.

On success, self.user is set to a Presenter or Student depending on the credentials of the user.

On failure, self.user remains None.

Parameters:
  • username - the username of the user
  • password - the password of the user

Precondition: self.user == None
