Class DrawingToolsModel

source code

A DrawingToolsModel holds the functionality of selecting any of the drawing tools from the DrawingTools tool bar. In our implementation, drawing is done on a frame known as a layer. Each user has a series of layers to draw on. See the Layer class under Presentation.py for more information.

Author: Mike Sevilla (mjsevill@calpoly.edu)
