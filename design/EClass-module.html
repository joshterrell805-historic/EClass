Package EClass

source code

EClass is a presentation tool which allows students to interactively follow a presentation by taking notes on the presentation itself, ask the presenter questions, rate how well they understand the presentation at any given moment, and converse in a forum with other students.

There are three types of applications that interact in EClass: EClass (presenter logged in), EClass (student logged in), and the central server. Documentation about how these applications interact behind the scenes is described in the networking documentation. Also, see EClass.Connection below to see the model of the EClass side of the connection.

Author: Josh Terrell jmterrel@calpoly.edu

  • EClass.LayerObject: Package LayerObject defines objects and operations within a presentation.
  • EClass.Person: Package Person defines objects and operations related to any user of the EClass.
  • EClass.Presentation: Package Presentation defines objects and operations related to the presentation's content and navigation.
  • EClass.View: Package View defines all views and GUIs that correspond to the EClass model.
