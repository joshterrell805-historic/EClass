Class MenuBar

source code

InitLocalVars(self, event)

source code 

Creates and returns a list of all IDs for each menu and menu item.

On success, the function returns a list of all IDs

On failure, the function does not return a list of all IDs

Parameters:
  • event - Event that calls this method.

OpenPresentation(self, event)

source code 

Opens the window allowing the user to open a pre-made presentation into the EClass program.

On success, the EClass program opens the .html file.

On failure, the EClass program does not open the selected .html file.

Parameters:
  • event - Event that calls this method

SavePresentation(self, event)

source code 

Opens the window allowing the user to save the current presentation as either a new presentation or overwrite the existing presentation.

On success, the EClass program saves the .creampie file.

On failure, the EClass program does not save the selected .creampie file.

Parameters:
  • event - Event that calls this method
