2. Functional Requirements

The primary function of the EClass system is presenting an interactive presentation where students and presenters can draw and communicate using an electronic whiteboard. EClass operations are performed by students and presenters on their personal whiteboard layers.

Following an overview of the EClass user interface, details of system use are presented in the following sections:

  • Preparing a presentation
  • Presenting with Electronic Classroom
  • Viewing a presentation
  • Presenter and student interactions
  • Tool details
  • Drawing
  • Managing the roster
  • Remote access
  • Menu bar details
These sections cover all functional requirements for the system.
