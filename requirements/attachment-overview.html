2.7.6. Attachment Tool

To use the attachment tool, the user moves the cursor over to the 'Drawing Tools' window and clicks on the attachment tool icon. As shown in Figure 1, once the attachment tool is clicked, the system responds by highlighting the tool and displaying the name of the tool in small text under the cursor.

Figure 1: Attachment tool icon.


Once the attachment tool is selected, EClass brings up a window, shown below in Figure 2, where the user can upload an attachment like an image. Figure 3 shows an image uploaded to the whiteboard by the user.

Figure 2: Selecting an object to upload.
Figure 3: Object is uploaded to whiteboard.
