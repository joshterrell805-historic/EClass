2.1. User Interface Overview

This section pertains to the general user interface of the EClass as well as both specific users: the presenter and the student. While both users share many of the same interfaces, there are also a few core differences. File, Edit, and View are menu bar interfaces that both users have access to. Presenter UI and Student UI are specific interfaces for each user with some overlap.
