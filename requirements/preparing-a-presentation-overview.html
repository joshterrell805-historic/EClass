2.2. Preparing A Presentation

Before a presentation can be presented, it first must be prepared. EClass uses XHTML to encode presentations, and through importing it allows existing HTML and XHTML documents to be used as the starting point for a presentation.

Presentations are more than plain HTML. There are special elements in EClass such as the slide and collapsable list elements.
