1.5.4. Saba Virtual Classroom Software

Saba Virtual Classroom Software (SVCS) is an interactive classroom software with a wide variety of supported features. While it is specifically titled for classrooms, its features are generalized enough to support a number of various online meetings and conferences. SVCS appears to be relatively unknown but offers an impressive suite of interactive features on multiple platforms including computers (via Web browser), mobile phones, and tablets.

Overall, SVCS is a viable option for hosting online classroom sessions and video conferences.

Good Features:

  • Includes the ability to raise a hand, leave the session, and invite someone else to the session
  • Mostly clean user interface with a number of abilities represented by icons
  • List of attendees with relevant options by their names
  • VoIP option means attendees can communicate verbally and not rely solely on text
  • Real-time polls that return results right away
  • Record classroom sessions and edit or manage them later
  • Access to all class material that was made available during a recording
  • Review graphical reports for class attendance, student polls, and in-lecture questions
  • Virtual whiteboard that everyone is able to write on

Bad Features:

  • "Current Room" option and ability to join multiple classrooms at once, which does not line up with the focus of the software described in these requirements
  • Video panel for an unspecified number of users, which adds clutter to the screen while open (Better option is only allowing the instructor and/or a single student to show up)
  • List of attendees is visible to everyone, which is unnecessary
  • Selectable "applaud" and "laughter" buttons
  • Ambiguous icon for leaving the classroom
  • A toolbar including all ability icons but with the count of students under each icon

Missing Features:

  • Instructor option to revoke drawing/writing privileges
  • Instructor option to remove someone from the class session
  • Ability to create slides by uploading a written document
  • Expandable slide headings and dynamic slide fitting
  • Custom instructor console (currently it looks very similar to the attendee's window with two or three extra buttons on the main side panel)
  • Multiple whiteboard layers, public and private layers
  • Session-long voting meters/sliders
  • Saving custom whiteboard content for students

Noteworthy features:

  • Students are able to enter virtual "breakout rooms" for group exercises
