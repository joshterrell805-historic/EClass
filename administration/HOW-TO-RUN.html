HOW TO RUN EClass




  1. Make sure Python version 2.7.X is installed. This is the version our team used and we know wxPython is supported in
  2. Make sure wxPython version 3.0 is installed. This is the version our team used and we know is supported by Python 2.7.X
  3. The path where the program is to be ran from is: http://users.csc.calpoly.edu/~kle17/EClass/implementation/source/python/
  4. To run the program simply run the command: python main.py

For convenience, here is a link to our documentation: EClass Doc
Also a note for the documentation, @author tags are only included in the documentation files and not the source files. We discussed this with you in a meeting during class.

One other thing currently is that when opening HTML documents on Apple machines, if the directory or filename has a space in it the file will not be loaded. This is a known issue and we are looking for a solution. One workaround is to load files from the root directory or User/.
