HOW TO RUN EClass




  1. Make sure Python version 2.7.X is installed. This is the version our team used and we know wxPython is supported in
  2. Make sure wxPython version 3.0 is installed. This is the version our team used and we know is supported by Python 2.7.X
  3. Make sure twisted is installed https://twistedmatrix.com/trac/wiki/Downloads
  4. Once twisted is installed, make sure twisted install path is on your PYTHONPATH. It's in /lib/.... do a find to find it
    1. `sudo find / -name twisted`
    2. `export PYTHONPATH=$PYTHONPATH:/System/Library/Frameworks/Python.framework/Versions/2.7/Extras/lib/python` is what worked on Andrew's mac
    3. NOTE: it's not ../lib/python/twisted ... it's .../lib/python
  5. Make sure webkit is installed (should be pre-installed on mac)
  6. The path where the program is to be ran from is: http://users.csc.calpoly.edu/~kle17/EClass/implementation/source/python/
  7. In order to log in, the server must be running. To run the server, run python main.py server from a separate terminal
  8. To run the program simply run the command: python main.py
  9. To authenticate, your username is `gfisher` and no password is required.

For convenience, here is a link to our documentation: EClass Doc
