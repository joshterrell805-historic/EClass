HOW TO RUN TESTS for EClass




  1. Python's 'unittest' module is used so no download should be necessary.
  2. Navigate to the testing/implementation/unit directory.
  3. Run the test.sh script
  4. View the test results directly in the console or terminal where you ran the test script.
